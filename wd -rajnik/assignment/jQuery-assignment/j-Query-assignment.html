1. What is jQuery?

jQuery is a JavaScript library.
The purpose of jQuery is make it easie to use JavaScript in website.
jQuery takes many common lines of JavaScript code, and wraps them in jQuery methods, that you can call in single line of code.

2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

addClass() - Adds one and more classes in selected elements Using j-Query.
removeClass() - Removes one and more classes from selected elements Using j-Query.
css() - Apply CSS properties in selected elements Using j-Query.
The jQuery animate() method is used to create animations Using j-Query.
